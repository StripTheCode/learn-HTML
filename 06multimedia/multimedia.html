Multimedia in HTML page

Multimedia elements in HTML allow us to add images, audio, and video to web pages, making them more interactive and engaging.

Different type/s of Multimedia

  1. Image
  2. Audio
  3. Video
  4. SVG (Scalable Vector Graphics)
  5. CANVAS
  6. Embed using iframe
    7.

Image :

used tag : 
<img src="html-5-logo.JPG" alt="html-lgo" width="100px" height="100px">

Audio :

used tag : 
<audio controls> <source src="audio.mp3" type="audio/mp3"> </audio>

Video :

used tag : 
<video width="320" height="240" controls> <source src="movie.mp4" 
        type="video/mp4"> </video>

SVG :

used tag : 
<svg width="100" height="100"> <circle cx="50" 
        cy="50" r="40" stroke="black" stroke-width="3" fill="red" /> </svg>

CANVAS :

used tag : 
<canvas id="myCanvas" width="200" height="100" style="border:1px 
        solid #000000;"></canvas>

iframe :

used tag : 
<iframe src="https://www.youtube.com/"
        width="460" height="265" frameborder="0" allowfullscreen></iframe>
